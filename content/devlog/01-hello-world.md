If you're reading this, Ophelia shipped. or it didn't and you found this in some graveyard of side
projects so either way, hi.

You might be early to reading this or used some time machine to look back at the start of the project,
and ask yourself — what is this website?
_"this looks like a template. the app doesn't even exist yet."_

This project was originally intended to be a simpler download manager so I could learn Rust.
I ended up working through it with a friend who actually knows low-level and networking code,
basically learning as we went.
As days went on, I stopped working on the project. I kept making static
websites and realized my love for making pretty UIs. Then I went back to
look at this project and realized "If I make a UI that I actually want to
work on, it'll be the thing that makes me want to work on this project"

I made a logo after watching some Youtube videos on how making logos is
just math, and since I'm majoring in math-related field (Physics) I
thought I'd give it a shot. I made something i really liked, gave it a
name that actually meant something to me, got excited and put this
website together in about a day.

> Ro, if you're reading this, love you brother

Over the next few weeks I wanna work on this non-stop.
I'll build this for me first. If it ends up being useful to you too,
I'm glad it did its job.
